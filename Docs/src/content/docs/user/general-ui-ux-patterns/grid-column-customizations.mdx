---
title: Grid - Column Customizations
---

Most grids in application are developed for user customizations in mind.

## Column Chooser
User can decide which column should be visible inside grid (and which not). Column chooser is possible to operate in two modes - Drag'n'Drop and Select

| Drag'n'Drop mode                                                                                                                                  | Select mode                                                                                                                                                    |
| ------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <img src={uiUxPatternGridColumnChooserDragndrop.src} alt="ui-ux-pattern-grid-column-chooser-dragndrop-link" />                                    | <img src={uiUxPatternGridColumnChooserSelect.src} alt="ui-ux-pattern-grid-column-chooser-select-link" />                                                       |

Column chooser button is located at upper-right corner of grid and has the following icon:

<img src={uiUxPatternGridColumnChooserButton.src} alt="ui-ux-pattern-grid-column-chooser-button-link" /> 

:::tip
All column choosers for grids in application are configured for **Select mode**.
:::

## Column Reordering
Column reordering is enabled on grids and it's available with drag'n'drop functionality when dragging column header.
<img src={uiUxPatternGridColumnReorder.src} alt="ui-ux-pattern-grid-column-reorder-link" />

## Column resizing
Column resizing is enabled on grids across all application.
<img src={uiUxPatternGridColumnResize.src} alt="ui-ux-pattern-grid-column-resize-link" />
