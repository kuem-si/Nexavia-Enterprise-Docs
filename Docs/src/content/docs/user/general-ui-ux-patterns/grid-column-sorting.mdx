---
title: Grid - Column Sorting
---

Most grid columns in application have column sorting feature enabled.<br />
:::caution
Not all columns have enabled sorting feature mainly because of technical reasons (e.g. agregated data in column) of database querying on the backend side.
:::

How column sorting looks like (example):

<img src={uiUxPatternGridColumnSorting.src} alt="ui-ux-pattern-grid-column-sorting-link" />

To apply sorting on column, click on column hader or right-click on column and select command from the context menu.

<img src={uiUxPatternGridColumnSortingContextMenu.src} alt="ui-ux-pattern-grid-column-sorting-context-menu-link" />

:::caution
Multi-column sorting in grids is mostly disabled due to technical reasons (e.g. excessive database queries on the backend).
:::
