---
title: Grid - Master-Detail
---

Some grids can contain larger sub-section of data per row. This type of grid is called **master-detail** grid.

The example picture below example of master-detail grid with description.

<img src={uiUxPatternGridMasterDetail.src} alt="ui-ux-pattern-grid-master-detail-link" />

Detail part of a specific row can be expanded or shrunk by clicking on the arrow icon at the beginning of the row.

Orientation of the row indicates if details are expanded (the arrow icon points to the bottom) or hidden (the arrow icon points to the right).

:::tip
Multiple row details can be expanded at the same time.
:::
