Each user can switch language before login.

<img src={loginPageSwitchLanguage.src} alt="login-page-switch-language-link" />

When user is authenticated, switching language can be done in the bottom right corner of right sidebar of application.

<img src={switchLanguageAuthenticatedUser.src} alt="switch-language-authenticated-user-link" />

<br />

:::tip
When a user is logged into the application, the chosen language is saved to the database as one of the settings on the user profile.
:::
